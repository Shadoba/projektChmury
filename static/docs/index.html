    Dokumentacja web API

    Formularz 'Add person'

    Odpowiada ze dodawanie nowej odoby do bazy danych. Pozwala nadać osobie następujące atrybuty: imię, nazwisko i kraj pochodzenia

    Tabela 'All people'

    Wyświetla wszystkie rekordy z bazy danych typu osoba

    Tabela 'Relations'

    Wyświetla wszystkie osoby znajdujące się w żądanej relacji względem żądanej osoby

    Formularz 'Link people'

    Odpowiada za tworzenie relacji między rekoradmi typu osoba (Person)

    Formularz 'Get relations of people'

    Odpowiada za wyświetlanie się rekordów w tabeli 'Relations'

    Przycisk 'Refresh'

    Odpowiada za aktualizacje rekordów w tabeli 'All people'
    UWAGA Aby odświerzyć listy rozwiajane, należy odświrzyć stronę w przeglądarce
